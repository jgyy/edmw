Well the first post certainly didnt talk about having different URLs. If that is the case, it is also very simple.

Simply replace the curl command with the following variant

Code: [ -f /tmp/download.url ] && curl -s -L -O `cat /temp/download.url`
If it is a web interface that determine which url to use, just write that url into /tmp/download.url.

The technique above will work with a lot of ways to trigger and shell script with the abundance of unix tools has so much power to adapt and integrate. You can even read from a JSON file if you like with tools like *jq*
