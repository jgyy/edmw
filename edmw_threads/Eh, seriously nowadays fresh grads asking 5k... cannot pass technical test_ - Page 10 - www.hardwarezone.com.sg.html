nubitol wrote:
What if the download takes more than an hour?
What so hard, just use a lock file to prevent re-entrant

Code: cd /dir/to/store/downloaded/file while true; do if [ ! -f /var/run/mylockfile ]; then (touch /var/run/mylockfile; curl -s -k -L -O https://host.name/file/path; rm -f /var/run/mylockfile) & sleep 3600 else sleep 1 fi done
The bad thing about the loop above is it will skew over a period of time, starting a child process helps to slow down the skew, but it will still eventually skew. Cron will be a better approach
